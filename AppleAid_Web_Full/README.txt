
AppleAid - Full Web Version (Voice + Assistant + Diagnostics)

✔️ Features:
- Voice-to-text question capture (mic)
- Real-time chat assistant with OpenAI (via API)
- Basic system diagnostics (browser battery, platform)

📦 Files:
- index.html — complete interactive assistant UI
- logo.png — placeholder (replace with your logo)

🌐 How to Deploy:
1. Upload to GitHub
2. Deploy on Vercel (or any static hosting)
3. In Vercel settings, add this ENV VAR:
   - OPENAI_API_KEY = your-secret-key-here

📌 Notes:
- This is frontend only (no Firebase used yet).
- Firebase can be added later for storing queries, auth, etc.
